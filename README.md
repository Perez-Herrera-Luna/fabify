# Fabify
Flesh and Blood is a trading card game from New Zealand studio Legend Story Studios. Fabify is a
Firefox and Chrome browser extension for replacing typed symbols with corresponding icons on the official website (fabtcg.com).

# Installation
You can find Fabify on the Chrome Web Store or Firefox Add-ons Marketplace
### Chrome
https://chromewebstore.google.com/detail/fabify/codfdeibdmdpjkekfihajfjdnlihmcpp
### Firefox
https://addons.mozilla.org/en-US/firefox/addon/fabify/
